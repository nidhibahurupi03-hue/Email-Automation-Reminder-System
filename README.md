# 📧 Smart Email Automation System

A powerful Streamlit-based Email Automation platform with scheduler, live email sending, analytics dashboard, and logging system.

---

## 🚀 Features

- 📧 Live Email Sending using SMTP (Gmail)
- ⏰ Automated Email Scheduler
- 📊 Analytics Dashboard (Plotly charts)
- 🧾 Contact Management System
- 📩 Email Preview System
- 📁 Logging (Sent / Failed emails tracking)
- 🚀 Dry Run + Live Mode support

---

## 🛠 Tech Stack

- Python 🐍
- Streamlit 🎯
- Pandas 📊
- Plotly 📈
- SMTP (Gmail Email Service)
- APScheduler ⏰
